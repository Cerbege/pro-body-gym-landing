# Probody

Probody is a simple front-end project built using **HTML5** and **CSS3**.
The project focuses on clean layout structure and basic visual effects.

## 🚀 Features

* Modern **navbar** design
* Clean and centered **hero section**
* Smooth **hover effects** for interactive elements
* A subtle **automatic animation** running on the page

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📱 Responsiveness

This project is **not responsive** and is designed for basic layout practice.

## 🌙 Dark Mode

Dark mode is **not included** in this version.

## 🎯 Purpose

The goal of this project is to practice:

* Layout structuring
* Styling with CSS
* Basic animations and hover interactions
